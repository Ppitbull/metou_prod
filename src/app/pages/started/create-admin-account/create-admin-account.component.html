Compte administrateur

Etape 2/4

Le nom du compte administrateur est obligatoire
La taile du nom de compte administrateur doit avoir au minimum 2 charactéres
Le nom de l'auto-école est obligatoire
Le nom de l'auto-école doit avoire au minimum 6 charactéres
Le nom de l'auto-école est obligatoire
L'email de l'administrateur doit respecter le format demandé
Example: vous@example.com
Le mot de passe de l'administrateur est obligatoire
Le mote de passe de l'administrateur doit avoir au moins 6 charactéres
Le mot de passe doit être saisi une seconde fois
Les deux (2) mots de passe doivent être similaire
Le numéro de téléphone de l'administrateur est obligatoire
Le numéro de téléphone de l'administrateur doit respecter le format demandé
Example: +237 698 29 53 68
